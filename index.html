Описание проекта

Проект создается для объединения большинства существующих мессенджеров в единую децентрализованную платформу для общения и обмена контентом в реальном времени

Цель нашего проекта

Долгосрочная задача проекта заключается в том, чтобы стать основной системой для отправки сообщений и синхронизации данных по всей сети, позволяя людям легко общаться друг с другом, давая пользователям контроль над своими данными и выбор между сервисами и вендорами, которые они захотят использовать
